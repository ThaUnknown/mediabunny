# @mediabunny/extra-codecs

Extra audio codec decoders for [Mediabunny](https://mediabunny.dev/): DTS, Dolby TrueHD, AC-3, E-AC-3, FLAC, and multichannel Opus.

Powered by [FFmpeg](https://ffmpeg.org/) compiled to WebAssembly.
